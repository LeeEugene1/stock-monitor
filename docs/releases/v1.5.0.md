# v1.5.0 - 2026-04-21

## CNN 공포·탐욕지수(Fear & Greed Index) 그래프 추가

### Added
- **CNN Fear & Greed Index** 시계열 차트를 시세 탭 인사이트 배너에 추가
  - 현재 지수값(0~100) + 라벨(극도의 공포 / 공포 / 중립 / 탐욕 / 극도의 탐욕) 표시
  - 최근 1년치 일별 히스토리 라인 차트 (recharts)
  - 5단계 컬러 밴드(Reference Area)로 심리 구간 시각화
  - 1주 전 / 1개월 전 / 1년 전 비교값
- 서버: `GET /api/market-insight/fear-greed` 엔드포인트
  - `production.dataviz.cnn.io` 프록시 + 1시간 인메모리 TTL 캐시
  - CNN 장애 시 stale 캐시 → 빈 페이로드 순으로 fallback (클라이언트 차트 자동 숨김)
