# v1.5.0 - 2026-04-21

## CNN 공포·탐욕지수(Fear & Greed Index) 도입

### Added
- **CNN Fear & Greed Index** — 시세 탭 인사이트 배너에 통합
  - 배너 스코어 행의 "시장 심리" 자리를 **탐욕지수** 막대바(극도의 공포 / 공포 / 중립 / 탐욕 / 극도의 탐욕)로 교체
  - "탐욕지수 그래프 (CNN Fear & Greed)" 토글(**기본 닫힘**) — 열면 1년치 일별 시계열 라인 차트 + 1주·1개월·1년 전 비교값 표시
  - recharts 기반, 5단계 컬러 밴드(ReferenceArea)로 심리 구간 시각화
- 서버: `GET /api/market-insight/fear-greed` 엔드포인트
  - `production.dataviz.cnn.io` 프록시 + 1시간 인메모리 TTL 캐시
  - CNN 장애 시 stale 캐시 → 빈 페이로드 순으로 fallback (클라이언트 UI 자동 숨김)

### Changed
- 시세 탭 배너 4번째 스코어 칸: "시장 심리" (VIX 기반 -2~+2) → "탐욕지수" (CNN 0~100 막대바)
  - 서버의 VIX 기반 `scores.fear`는 종합 시그널/요약 계산에서 계속 사용됨 — UI 표시만 교체

### Technical
- 새 컴포넌트: `client/src/components/market-insight/FearGreedChart.tsx`
  - `FearGreedScoreItem` — 기존 ScoreBar와 동일 레이아웃의 막대 칩 (숫자 대신 라벨만)
  - `FearGreedDetails` — 토글 + 차트 래퍼 (차트는 열렸을 때만 마운트)
  - `useFearGreed` 훅으로 React Query 캐시 공유 → 두 컴포넌트가 요청 1회
- 서버 타입 추가: `FearGreedData`, `FearGreedPoint`, `CnnFearGreedResponse`
- CNN 엔드포인트는 `Origin`/`Referer` 없이는 418 → 브라우저 헤더 세트로 요청
