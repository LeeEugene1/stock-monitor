# v1.4.0 - 2026-04-21

## 키움 매수 주문 API + 자동매수 브로커 분기

### Added
- `KiwoomService.orderCash`: 키움증권 현금 매수 주문 API (`kt10000`)
  - KIS와 동일한 시그니처, `OrderResult` 반환
  - 주문 유형 매핑: `ordDvsn` → `trde_tp` (시장가/지정가/조건부지정가)
  - 키움 에러 응답(`return_code`) 파싱 → 명확한 에러 메시지
- 자동매수 규칙 목록에 브로커 뱃지(한투/키움) 표시 — 계좌 이름 옆

### Fixed
- **자동매수 스케줄러가 키움 계좌에서 403 에러로 실패하던 문제**
  - 스케줄러가 브로커 무관하게 `KisService.orderCash`만 호출 → 키움 앱키로 KIS 서버 접근 불가
  - `account.broker === 'kiwoom'`일 때 `KiwoomService.orderCash`로 분기
- AutoBuyModule에 KiwoomModule import 누락 수정
