# v1.0.0 - 2026-04-12

## 멀티계좌 포트폴리오, 자동매수, 시장 인사이트

### Added
- SQLite DB + TypeORM 기반 데이터 영구 저장
- 한국투자증권(KIS) OpenAPI 연동 (토큰 발급/캐시, 잔고 조회, 주문)
- 멀티계좌 포트폴리오 대시보드 (전 계좌 합산 + 계좌별 상세)
- 자동매수 스케줄러 (매월 N일, 시장가 주문, cron 기반)
- 시장 인사이트 (NASDAQ/S&P500/VIX/USD-KRW/US10Y 매일 수집, 종합 시그널)
- 관심종목 DB 저장 (localStorage → SQLite 마이그레이션)
- 계좌 관리 CRUD UI (앱키 마스킹, CORS 제한)
- react-router-dom 라우팅 (시세/포트폴리오/자동매수/계좌관리)
- @tanstack/react-query 서버 상태 관리

### Security
- API 응답에서 appKey/appSecret/accountNo 마스킹
- CORS를 localhost만 허용
