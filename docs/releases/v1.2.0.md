# v1.2.0 - 2026-04-16

## 키움증권 연동, 자동매수 고도화, 알림 시스템

### Added
- 키움증권 OpenAPI 연동 (토큰 발급, 잔고/요약 조회)
- 알림 시스템 (매수 시점/성공/실패 알림, NotificationBell UI)
- 자동매수 스케줄 확장 (daily/weekly/monthly, 트리거 조건, 윈도우 시간대)
- 자동매수 주문 유형 (시장가/지정가, 금액 전략: 고정/예수금 비율)
- `useStockPrices` 훅 (자동매수 규칙 목록에서 실시간 현재가 표시)
- `formatKRW`, `BROKER_LABELS`, `PRODUCT_LABELS` 공통 유틸/상수 추출

### Changed
- 포트폴리오에서 KIS/키움 브로커 자동 분기
- 키움 `inquireBalanceAndSummary`로 중복 API 호출 제거
- 자동매수 cron을 평일 장시간대(08-15시)로 제한
- `evaluateTrigger` → `executeAutoBuy` 간 현재가 데이터 공유 (중복 스크래핑 제거)
- `AutoBuyRuleList` codes/accountMap `useMemo` 적용

### Fixed
- `notification.service.ts`: `readAt: null as any` → `IsNull()`
- `AutoBuyLogTable` early return 전 불필요한 Map 생성 제거
