# v1.3.0 - 2026-04-18

## 포트폴리오 시각화, 브랜딩 개선

### Added
- 포트폴리오 카테고리별 비중 파이차트 (recharts)
- `CategoryModule`: 네이버 업종 자동 수집 + 사용자 태그 오버라이드 (하이브리드)
- `GET /api/categories`, `PUT /api/categories/:stockCode`
- 파이 슬라이스/범례 클릭 → 카테고리명 편집 모달
- 공용 유틸: `server/src/common/naver-http.ts` (NAVER_HEADERS)

### Changed
- 브랜딩: 페이지 타이틀 "Vite + React + TS" → "Stock Monitor", 파비콘을 주식 차트 SVG로 교체
- `formatKRW`: 만/억 단위에도 콤마 포맷팅 (6,096만원)
- `USD/KRW`, `환율` 표시에 콤마 적용
- MarketInsightBanner에서 스코어 바 섹션 제거 (간결한 뷰)

### Removed
- Dead code: `Scores` 인터페이스, `ScoreBar` 컴포넌트, `.score-*` CSS, `resolveCategory` 미사용 메서드

### Fixed
- `Portfolio.css` 모달 스타일이 전역 `.modal-*` 규칙과 충돌해 `StockDetailModal` 회귀 유발 → 제거 후 전역 재사용
- `CategoryService.resolveCategoriesBatch`: OR 체인 where → `In()` 연산자로 교체
- `fetchSector`: `$('*').each` 전체 DOM 순회 → 구체 CSS 셀렉터 + 정규식 fallback
- 네이버 스크래핑 동시성 5개로 제한 (rate-limit 방지)
- `CategoryEditModal`: 변경된 항목만 저장, 에러 처리 추가
- `computeBreakdown`: `evalAmount <= 0` 종목 필터링
- `PortfolioBreakdown`: `chartData`/`TOOLTIP_STYLE` 메모이제이션으로 불필요한 리렌더 방지
